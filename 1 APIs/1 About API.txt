API, REST, HTTP Metodları və Endpointlər Haqqında Sadə Təlim Materialı


API nədir, REST nədir, internetdə proqramlar necə "danışıb" bir-biri ilə məlumat mübadiləsi edir. 
Bu mövzular bir az texniki səslənə bilər, amma mən onları sadəcə, gündəlik həyat nümunələri ilə 
izah edəcəyəm. Təsəvvür edin ki, kompüterlər və proqramlar bir-biri ilə "qonşu evlər" kimi əlaqə 
qurur. Bir proqram digərindən məlumat istəyir, digəri cavab verir. Bu əlaqəni necə quracaqlarını 
öyrənək. Addım-addım gedək, hər şeyi yavaş-yavaş başa düşəcəksiniz. Əgər bir yer qarışıq gələrsə, 
sual verin!


1. API Nədir? (Application Programming Interface)

API-nin tam adı Application Programming Interfacedir. Bu, Azərbaycan dilində Tətbiq Proqramlaşdırma 
İnterfeysi kimi tərcümə olunur. Sadə dillə desək: API, proqramların bir-biri ilə "danışması" üçün 
qaydalar və yollar təklif edən bir köprüdür. Təsəvvür edin ki, siz dostunuzla telefonla danışırsınız. 
Siz bir sual verirsiniz ("Bu gün hava necədir?"), dostunuz cavab verir ("Yağışlıdır"). API də eyni belədir: 
Bir proqram (məsələn, sizin mobil tətbiqiniz) digər proqrama (məsələn, hava proqnozuna) sual verir (Request) 
və cavab (Response) alır.



Niyə API hər yerdədir?

- API-lər həyatımızın hər yerindədir! Məsələn:

    - Instagram-da şəkil paylaşanda, Facebook API-si vasitəsilə dostlarınıza göndərilir.
    - Google Xəritələrində yol təsviri istəyəndə, bu API vasitəsilə başqa proqramlardan məlumat gəlir.
    - Bank tətbiqində balansınızı yoxlayanda, bankın API-si işləyir.


- API, bir proqramın digərinə "sözləşmə" təklif etməsidir. O, deyir: "Mən sənə məlumat verəcəm, amma sən 
mənim qaydalarıma uy. Mənə belə soruş, mən belə cavab verəcəm."





API necə işləyir? (İstək və Cavab Strukturu)

    - İstək (Request): Siz bir proqrama soruşursunuz. Məsələn, "Mənə istifadəçiləri göstər" deyirsiniz. Bu, soruğu URL (veb ünvanı) vasitəsi ilə göndərilir.
    - Cavab (Response): Proqram cavab verir. Məsələn, istifadəçilərin siyahısını JSON formatında (sadə mətn kimi) geri göndərir.
    - Nümunə: Təsəvvür edin ki, bir kitab mağazası tətbiqi. Siz "Kitabları göstər" deyirsiniz (istək). API, kitabların siyahısını (ad, qiymət, təsvir) göndərir (cavab).
